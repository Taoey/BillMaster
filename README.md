# BillMaster


## 前言

个人账单统计系统（单机版）。

由于账单是比较隐私的信息，本人认为放到任何一个三方服务器上都不安全，故该系统只做成单机版，用户导入的微信/支付宝账单只存在于自己的电脑上，不会上传到任何三方服务器上，请放心使用



## 主要功能

- [ ] 支付宝/微信账单导入
  - [ ] 文件上传方式导入
  - [ ] 邮箱上传
- [ ] 账单搜索查询
- [ ] 账单可视化统计
- [ ] 账单标签
- [ ] 账单自动化标签
- [ ] 账单日报/周报/月报
- [ ] 发票管理
- [ ] 账单详情（支持图片）
- [ ] 收支计划
